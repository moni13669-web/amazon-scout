Use Excel or CSV
Any .xlsx, .xls, or .csv file works. One sheet is enough.

The first row must have a column header exactly named ASIN (uppercase). Other columns are ignored.

Each cell under ASIN must contain exactly one 10-character ASIN. Invalid rows are skipped automatically.
